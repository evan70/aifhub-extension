# Roadmap проекта

## Сводка

AIFHub Extension — расширение для ai-factory 2.x CLI со spec-driven workflow. Включает навыки анализа проекта, создания структурированных планов и управления артефактами. Pure skills extension без исполняемого кода — только Markdown-инструкции для AI-агентов.

**Сильные стороны:** Архитектура, документация, CI/CD, workflow structure
**Критичные пробелы:** Тестирование
